## 💰 Money Tracker App
A Flutter mobile application for managing personal income and expenses, integrated with Supabase as the backend service.

## 📱 Features
- Add income and expense records
- View transaction history (combined income + expense)
- Display total income, total expense, and balance
- Auto-calculated financial summary
- Delete-ready architecture (future upgrade)
- Splash screen + welcome screen flow
- Clean and modern finance UI (Gradient + Glass style)

## 🧰 Tech Stack
- Flutter
- Dart
- Supabase
- Intl (currency & date formatting)
- Material Design
